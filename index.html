Descripción del Proyecto

Este proyecto consiste en un sistema de registro de ventas mensuales de computadores por marca, desarrollado en Java con Maven.

El programa permite:

  • Iniciar sesión con usuario y contraseña (guardados en un archivo .txt)
  • Ingresar una matriz de ventas (meses × marcas)
  • Calcular promedios por fila, columna y total
  • Encontrar valor más alto y más bajo
  • Guardar resultados en un archivo .txt
  • Subir los datos en formato JSON a GitHub (se actualiza cada vez que se usa)
